# E-commerce Database Simulation

This project simulates an e-commerce marketplace database using Python and SQLite.

The objective was to design a relational database schema and populate it with synthetic but realistic data to mimic marketplace activity.

## Tools Used
- Python
- SQLite
- Pandas
- NumPy
- Faker

## Database Structure
The database includes several tables such as:

- Users
- Products
- Categories
- Orders
- Order_Items
- Reviews

## Key Features
- Synthetic data generation using Faker
- Realistic email generation logic
- Brand–product relationships
- Probabilistic order generation
- Relational database structure with linked tables

## Project File
`ecommerce_database_simulation.ipynb` – main notebook containing the full workflow.
